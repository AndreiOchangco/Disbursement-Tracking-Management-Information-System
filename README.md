# Disbursement Tracking Management Information System (DTMIS)

A web-based **Disbursement Tracking Management Information System** developed for the **Local Government Unit (LGU) of Luna, La Union** to improve the efficiency, accuracy, accessibility, and transparency of disbursement monitoring and financial record management.

This project was developed as part of the requirements for **ELECT COG 3 – Research Methods for IT** at **Saint Louis College**.

---

# Project Overview

Local government financial operations often rely on **manual documentation and spreadsheet-based monitoring**, which leads to delays, fragmented records, and difficulty retrieving financial documents.

The **Disbursement Tracking Management Information System (DTMIS)** addresses these problems by providing a **centralized digital platform** where disbursement transactions can be recorded, monitored, and managed through structured workflows.

The system allows authorized users to track the progress of financial disbursements from request submission up to the release of funds while maintaining transparency and accountability.

---

# Problem Statement

The current disbursement monitoring process within the LGU faces several operational issues:

- Reliance on **manual tracking and spreadsheets**
- Difficulty retrieving financial documents
- Delays caused by **manual follow-ups between offices**
- Lack of centralized financial records
- Manual processing of **disbursement vouchers and journal entries**

These limitations create inefficiencies in financial monitoring and reporting.

---

# Objectives

## General Objective

To design and develop a **Disbursement Tracking Management Information System** for the Local Government Unit of Luna to improve efficiency, accuracy, accessibility, and transparency in managing disbursement and financial records.

## Specific Objectives

- Enhance the existing **spreadsheet-based disbursement tracking process**
- Implement **structured workflows and automated tracking**
- Improve coordination among LGU offices
- Provide **role-based access control** for system users
- Reduce delays and manual follow-ups in financial processing
- Support **better monitoring of disbursement transactions**

---

# System Features

## Centralized Disbursement Records
All financial transaction records are stored in a **centralized database**, reducing duplication and improving accessibility.

## Disbursement Voucher Tracking
Allows users to monitor the status of disbursement vouchers through different stages of processing.

## Workflow-Based Transaction Monitoring
Transactions move through a defined workflow from request submission to fund release.

## Role-Based Access Control
System access is limited depending on the responsibilities of each office.

## Budget Monitoring
Tracks budget allocations and actual disbursement transactions.

## Financial Reporting
Generates reports that help management evaluate financial activities and decision-making.

## Audit Trail
System logs record actions performed by users for accountability and transparency.

# System Users

The system supports **six main user roles**:

| Role | Description |
|-----|-----|
| System Administrator | Manages system configuration and user accounts |
| Accountant | Records financial transactions and disbursement vouchers |
| Budget Officer | Monitors budget allocations and expenditures |
| Treasurer | Manages the release of funds |
| Secretary | Encodes and tracks disbursement documents |
| Technical Officer | Oversees system reports and transaction monitoring |

---

# System Workflow

The system tracks disbursement transactions through the following process:

1. Transaction Request Submission  
2. Encoding of Disbursement Documents  
3. Budget Verification  
4. Approval Process  
5. Fund Release  
6. Recording and Reporting

Transaction statuses may include:

- Pending
- Approved
- Released

---

# System Scope

The proposed system focuses on **monitoring and managing disbursement transactions** within the Local Government Unit.

Included functions:

- Disbursement voucher tracking
- Budget monitoring
- Petty cash monitoring
- Transaction status monitoring
- Financial reporting

---

# System Limitations

The proposed system is **not a full accounting system**.

The following functions are outside the scope:

- General Ledger Management
- Payroll Processing
- Revenue Collection
- Automated Financial Posting
- Integration with national government financial systems

Financial data must still be **manually entered by authorized users**.

# Research Methodology

The study used the following research approaches:

### Descriptive Research Design
Used to analyze the current disbursement monitoring process and identify operational challenges within the LGU.

### Developmental Research Design
Used to design and develop the proposed Disbursement Tracking Management Information System.

### Data Collection Methods
- Interviews with LGU personnel
- Review of financial records and documentation
- Analysis of current spreadsheet tracking methods
